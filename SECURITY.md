# Security Policy

## Supported Versions

Cells is pre-1.0. Security fixes are applied to the latest tagged release and, when needed, the current `main` branch.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| `main` | Yes |
| Older releases | No |

## Reporting a Vulnerability

Do not open public GitHub issues for security vulnerabilities.

Instead, email `r@raj.how` with:

- A clear description of the issue
- Affected version or commit
- Reproduction steps or a proof of concept
- Any suggested mitigation, if you have one

You should receive an initial response within 5 business days. After triage, the goal is to keep you informed about severity, fix status, and disclosure timing.

When possible, please give time for a fix to be prepared before public disclosure.
